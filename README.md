## Instalação e uso

Clone o repositório

```
$ git clone https://github.com/deniscoelho-js/portfolio_02.git
```

Siga os passos abaixo

```
# Instale as dependências
$ npm i or yarn

# Inicie o servidor
$ yarn server or npm run server 

# Iniciar o projeto
$ yarn start or npm run start
```
